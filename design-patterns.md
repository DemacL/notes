## 行为模式

### 策略模式 VS 模版方法模式
|    |     |   |   |   ||
|---|---|---|---|---|---|
| 策略模式  | 基于对象   | 动态  |  组合 | 封装可变的策略，强调的是选择不同的算法策略来实现同一目的   | 算法整体可选，定义不同的算法 |
|  模版方法模式 | 基于类  | 静态 | 继承   |  在于封装算法的流程，强调的是流程的统一性和稳定性  | 算法流程固定，自定义算法里面的特殊步骤 |
